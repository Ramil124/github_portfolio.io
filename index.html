Electricity Contract Selection using Python

After installation of the smart electricity meter. This model reports the usage to the electricity provider every hour. The electricity contract is due for renewal so being an analyst my job was to decide and find out which contract will minimize the annual electricity cost.
There are 3 different types of electricity contracts available:
• No Flex: The cost per kWh of electricity is constant for the entire year
• Monthly Flex: The cost per kWh of electricity fluctuates depending on the month
• Hourly Flex: The cost per kWh of electricity fluctuates based on the time of day
I have used an electricity usage dataset for the last year and cleaned the data with RegEx to make the data more readable. Then used Pandas to create a DataFrame for the values of the data and calculated the key performance metrics from the contracts that are presented.

Rock Formations with Python

The data that I am using is from the Drilling Operations which is available online. Feature values in this dataset are very carefully collected throughout the whole Drilling process and are quite indicative of the firmness of the formations being drilled. I have used this data to create a Machine Learning model that will help me to predict a Rock Formation during drilling operations. After selecting the features with no redundant values except for Weight on the Drill Bit,as it is one of the most important features that can tell us a lot about the Lithology around us,I have removed the Outliers from this column and prepared the data to be used for the Lithology prediction. This model can be very helpful in improving the drilling process as well as being helpful in terms of time and cost efficiency.

Leaf disease Detection
with Deep Learning

I have built a Deep Learning, to overcome these disadvantages and used a Convolutional Neural Network model for image recognition to achieve the detection of plant diseases faster and more accurately.
